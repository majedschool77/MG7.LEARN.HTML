نبذة عن HTML

HTML هي لغة ترميز تُستخدم لبناء هيكل صفحات الويب، وهي أساس كل موقع على الإنترنت.

HTML

  • HTML هي اختصار لـ HyperText Markup Language.
  • هي اللغة الأساسية لإنشاء صفحات الويب.
  • تستخدم لوصف هيكل ومحتوى الصفحة (نصوص، صور، روابط، جداول، إلخ).

أهمية HTML:

  • دون HTML، لن يتمكن المتصفح من عرض أي صفحة ويب.
  • HTML تتيح تنظيم المحتوى بطريقة واضحة للزوار.

أجزاء HTML الأساسية:

  • العناصر (Elements): مثل H1 للعناوين وP للفقرات.
  • لسمات (Attributes): لإضافة معلومات إضافية للعناصر، مثل href في الروابط.
  • لوسوم (Tags): تُكتب بين < > وتحدد بداية ونهاية العنصر.
